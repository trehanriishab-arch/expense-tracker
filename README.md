# Expense Tracker (Python CLI)

A simple command-line based expense tracker built using Python.  
It allows students to record, view, and manage daily expenses with dates.

## Features
- Add new expenses with amount and date
- View all expenses with total
- Delete any expense
- Data saved automatically to file
- Loads data on startup
